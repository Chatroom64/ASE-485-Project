### Learning with AI - Requirements
- As this is a side project, I want to keep the requirements as minimal as possible:
  - Create the logic of a wheel that contains list elements using TypeScript.
  - Render the wheel on the page using SVG.
  - User can create or delete a list with list elements.
  - User can insert that list into a wheel.
  - User can click a button to spin the wheel.
  - What the wheel lands on will be displayed on the page
